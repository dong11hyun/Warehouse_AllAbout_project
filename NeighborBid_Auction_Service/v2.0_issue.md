### 🔴 문제점 (As-Is)

**1. 문제 상황: HTTP와 WebSocket의 실행 프로세스 분리로 인해, 즉시 구매(HTTP 요청) 시 실시간 알림이 웹소켓 그룹으로   전송되지 않는 문제**
- **HTTP -> 웹소켓 :** 즉시 구매하기(HTTP) 눌렀을 때, 다른 사용자들에게 알림이 안오는 문제 **(웹-웹 끼리는 잘됨)**


    

### 🟢 고민과정 및 해결

    **`transaction.on_commit`을 적용해 DB 저장 후 확실하게 실시간 알림이 브로드캐스팅 되도록 개선**
