## <쿠X 판매자 메타 데이터 수집>
- playright와 chrome DevTools Protocol 활용한 비동기 웹 크롤러
- 쿠팡 검색 결과 페이지에서 상품리스트 수집, 각 상품 상세페이지 접근 후 판매자 정보
(상호,사업자번호,연락처) 등을 자동으로 추출하는 파이썬 봇.
- 봇탐지 시스템 회피를 위해 기생모드와 비동기 병렬 처리 아키텍처 적용
--- 
### <tech_stack>

1. python 
2. Playwright (Async)
    - 비동기 브라우저 제어 및 데이터 추출
3. Chrome CDP
    - 기존 실행된 크롬(9222 포트)에 연결하여 봇 탐지 회피
4. asyncio
    - gather, Semaphore를 활용한 병렬 크롤링 
5. CSV
    - asyncio.Lock을 적용하여 데이터 쓰기 충돌 방지
---
