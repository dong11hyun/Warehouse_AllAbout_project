## 확장성

**수집된 매장의 위도, 경도 추가 / 시각화를 위해 카카오맵 api 호출하여 지도 확인가능**
`http://127.0.0.1:8000/map/`

**시각화 강화**
pandas 로 상권들의 점수를 매겼고, 분석코드로 점수를 표로 산출함

> 향후 지도_API 에 매장별 점수 띄우기

---
[Backend] TargetStore 모델 생성 및 collect_cafes.py가 DB 데이터를 읽도록 수정. (가장 시급)
[Frontend] 간단한 ListView와 DetailView를 만들어 수집된 데이터를 웹 표로 보여주기.
[DevOps] Docker로 감싸서 배포 준비 및 PostgreSQL 전환.
[Performance] Celery 도입하여 수집 속도 및 UX 개선.
